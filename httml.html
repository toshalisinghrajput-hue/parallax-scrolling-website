INTRO TO HTML

HYPERTEXT MARKUP LANGUAGE

HTML is the standard markup language for creating Web Pages.
HTML describes the structure of Web Page.
Tim Berners-Lee invented HTML in the year 1991.
The latest edition of HTML is HTML5.2
HTML elements tells the browser how to display the content.
HTML consists of a series of elements.
HTML is primarily used for static websites.
HTML can be combined with CSS(cascading style sheets) and JS(java script).
